hello-world大集合

2020-03-28

学计算机的都需要向”hello world”致敬！！！
原本是hexo搭建后自主形成的一个基础模板，但想了想干脆把这个博文改成一个相对有趣的博文。都说hello world是只有程序员才明白的梗，那我就上网查一下把所有语言的hello world来个大集合！快来看看吧！这里有没有你熟悉的语言呢？

热门语言

Pascal

program Helloworld;
begin writeln(‘hello,world.’);
end

C

#include<stdio.h>
main( ){
   printf(“hello world!”);
}

C++

#include<iostream>
using namespace std;
int main( ){
  cout << “Hello World”<<endl;
  return 0;
}

Fortran

PROGRAM HELLOWORLD
PRINT *, ‘Hello, World!’
END

PERL

$welcome = “Hello World”;
print “$welcome\n”;

HTML

<html>
 <head>
  <title>Hello, World</title>
 </head>
 <body>
   <h1>Hello, World</h1>
 </body>
</html>

SQL

SELECT ‘HELLO, WORLD’ AS HELLO

JavaScript

alert(‘Hello, World.’);

PHP

C’#’

public class Hello
{
  public static void Main()
  {
    System.Console.WriteLine(“Hello, World!”);
  }
}

Java

public class Hello {
  public static void main(String[] args) {
        System.out.println(“Hello, World”);
  }
}

###Python

print(“hello world”)
